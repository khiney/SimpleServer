You can make the following GET requests

  • **NOTE** Do not include square brackets [] when entering values
  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request *not working :( *
  • /random -- returns index.html
  • /multiply -- enter two numbers, example: /multiply?num1=[int]&num2=[int]
  • /area -- enter two numbers, example: /area?heaight=[int]&base=[int]
  • /palindrome -- enter a word that's 3 letters or longer (no spaces) and I'll tell you if its a palindrome, example: /palindrome?check=[word]
